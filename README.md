## Overview

This repository contains the complete code, data, and results for the paper:

**"Impact of Technical Indicators on Machine Learning Models for Forex Market Prediction"**

We evaluate 11 technical indicators across 27 currency pairs using 10 years of daily data and 3 machine learning models (Logistic Regression, Random Forest, SVM) with 5-fold time series cross-validation.

## Key Finding

**Baseline (OHLC only) wins on all 27 pairs.** Technical indicators degrade predictive accuracy by an average of 8.10 percentage points.
